## レスポンシブ対応方針

| ブレークポイント | 対応 |
|----------------|------|
| mobile (<768px) | 打刻ボタン大表示、テーブルはカード形式に変換 |
| tablet (768-1024px) | サイドナビ折りたたみ |
| desktop (>1024px) | フル表示 |

- 打刻画面はモバイルファースト（大きなボタン、片手操作可能）
- 一覧画面はデスクトップではテーブル、モバイルではカードリスト
