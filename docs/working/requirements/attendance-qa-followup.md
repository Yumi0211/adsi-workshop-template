# 勤怠管理アプリ — 追加 Q&A

## ICカード連携

[Question] ICカードの種類は？（例: FeliCa/NFC、社員証に内蔵、専用カード）
[Answer]専用カード

[Question] カードリーダーの設置場所と台数は？（例: 各フロア入口に1台、各デスクにUSBリーダー）
[Answer]入口に1台

[Question] ICカード打刻に加えて、Web画面からの手動打刻（ICカード忘れ時のフォールバック）は必要ですか？
[Answer]Web画面からの手動打刻も可能にする

## SSO連携

[Question] SSO の IdP（認証基盤）は何ですか？（例: Microsoft Entra ID (Azure AD)、Okta、SAML汎用）
[Answer]Microsoft Entra ID (Azure AD)

## 有給休暇

[Question] 有給の付与ルールはシステムで自動管理しますか？それとも管理者が手動設定？
[Answer]自動管理

[Question] 有給の種類は？（例: 全休/半休のみ、時間単位有給あり）
[Answer]全休/半休のみ

## 残業・承認ワークフロー

[Question] 残業の承認フローは？（例: 事前申請→承認が必要、事後の実績ベースで承認不要、閾値超過時のみ承認）
[Answer]閾値（45時間/月）超過時のみ承認

[Question] 承認者が不在の場合の代理承認はありますか？
[Answer]なし

## 部門構造

[Question] 組織階層は何段ですか？（例: 会社→部→課の3階層、会社→事業部→部→課の4階層）
[Answer]会社⇒本部⇒部⇒課

[Question] 兼務（複数部門に所属）はありますか？
[Answer]あり

## モバイル対応

[Question] モバイル対応の範囲は？（例: レスポンシブWebで十分、ネイティブアプリ必須、PWA）
[Answer]レスポンシブWebで十分
